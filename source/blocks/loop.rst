循环
======


break
-------------

**描述：**   break 语句可以跳出 for 和 while 的循环体。如果你从 for 或 while 循环中终止，任何对应的循环 else 块将不执


continue
-------------

**描述：**   continue 语句被用来告诉 Python 跳过当前循环块中的剩余语句，然后继续进行下一轮循环
